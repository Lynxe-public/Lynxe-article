# AI Agent Articles Blog

A blog about ReAct Agent, Function Calling, MCP, and Skills, built with VitePress and deployed on GitHub Pages.

## Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions when you push to the `main` branch.

## Setup Instructions

1. **Update GitHub Pages settings:**
   - Go to your repository Settings → Pages
   - Source: GitHub Actions

2. **Configuration:**
   - Config is already set for `Lynxe-public/Lynxe-article`
   - Add your Google Search Console verification code if needed

3. **Move markdown files:**
   - Copy Chinese articles to `docs/zh/`
   - Copy English articles to `docs/en/`
   - Update image paths if needed

4. **Push to GitHub:**
   ```bash
   git add .
   git commit -m "Setup VitePress blog"
   git push origin main
   ```

5. **Enable GitHub Pages:**
   - The GitHub Action will automatically deploy your site
   - Your site will be available at `https://Lynxe-public.github.io/Lynxe-article/`
